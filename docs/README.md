# Dokumanlar

Bu klasor, `theapdesign.art` web sitesi, Del-It ve XOX Taktik Arena urun yayin surecleri icin guncel referans dokumanlarini icerir.

## Dosyalar

- `website-tasarim-gidisat.md`: Web sitesi sayfa yapisi, tasarim kararlari ve teknik durum
- `uygulama-tasarim-gidisat.md`: Del-It ve XOX uygulamalari, urun mesaji, App Store ve web sitesi entegrasyon notlari
- `proje-sureci-ve-operasyon.md`: Repo, build, deploy ve operasyon notlari
- `icerik-ve-teslim-kontrol-listesi.md`: Icerik, SEO, yasal sayfa ve deploy kontrol listesi

## Guncel Durum

- Ana sayfa, Uygulamalar, Del-It, XOX Taktik Arena ve Iletisim akislari mevcut.
- Del-It ve XOX icin gizlilik politikasi ve kullanim sartlari sayfalari mevcut.
- Simdilik yayinlanmayacak urunler web sitesinden kaldirildi.
- Site Cloudflare/Vite build akisi ile yayinlanir.
